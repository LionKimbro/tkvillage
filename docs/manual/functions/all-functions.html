declare_app

declare_app(spec)
module: tkvillage.appauthority: runtime declaration

Preferred application declaration and initialization API. It resets runtime state, resolves the project directory, optionally creates the hidden root, loads config, registers debug windows, and installs the debug shortcut.

Parameters

spec: dictionary with required keys name, project-dir-name, and shutdown-policy. Optional keys include project-root, tick-interval-ms, test-mode, create-root, shutdown-window-kind, and on-shutdown.

Returns

rt.g.

Things to Know

shutdown-policy is non-optional. Use this for new code; create_app() is deprecated.

create_app

create_app(name="tkvillage", project_dir_name=".tkvillage", project_root=None, tick_interval_ms=50, test_mode=False, create_root=True)
deprecatedauthority: runtime declaration

Compatibility wrapper around declare_app(). It emits DeprecationWarning and declares an explicit-shutdown runtime.

Use Instead

declare_app(spec).

request_shutdown

request_shutdown(reason=None)
authority: graceful shutdown request

Sets the explicit shutdown flag and optional reason. Shutdown begins during the tick-end policy check.

Side Effects

Updates rt.g["shutdown_requested"], rt.g["shutdown_reason"], and appends to rt.runtime_log.

reset_app

reset_app()
authority: runtime reset

Resets runtime state to defaults while preserving dictionary and list identities.

Things to Know

Destroys an existing root if possible. Uses clear() and update() rather than rebinding runtime containers.

run

run()
authority: Tk mainloop

Ensures the hidden root exists, starts the tick loop, and enters tkinter.mainloop().

Things to Know

This blocks until the Tk mainloop exits.

shutdown

shutdown()
authority: hard cleanup

Stops ticking, calls the shutdown handler once if configured, destroys live windows and root, and marks the runtime not running.

declare_config

declare_config(name, default=None, type="string", description="", choices=None)
authority: config schema

Declares a typed configuration key and initializes its current value if absent.

Parameters

type may be string, int, float, bool, path, or choice. choices is required for choice config.

load_config

load_config()
authority: config persistence

Reads config.json from the project directory and applies declared type coercions. Returns rt.config_values.

save_config

save_config()
authority: config persistence

Writes rt.config_values to project config.json as indented JSON. Path values are serialized as strings.

get_config

get_config(name)
authority: config read

Returns the current config value. Raises KeyError when the key is absent.

set_config

set_config(name, value, persist=True)
authority: config mutation

Sets, coerces, and optionally persists a config value. Undeclared keys are auto-declared with an inferred type.

list_config

list_config()
authority: config inspection

Returns declared config rows with current values included.

register_window_kind

register_window_kind(window_kind, title=None, multiplicity="singleton", create=None, make_initial_state=None, reduce_event=None, project=None, on_show=None, on_close=None, on_destroy=None, on_tick=None, debug_label=None)
authority: window-kind registry

Registers a summonable Toplevel window kind.

Callback Signatures

  • make_initial_state(key=None, payload=None)
  • create(record)
  • reduce_event(state, event)
  • project(record)
  • on_show(record), on_close(record), on_destroy(record), on_tick(record)

Things to Know

embedded and modal multiplicities are recognized but not implemented.

summon_window

summon_window(window_kind, key=None, payload=None)
authority: window creation/reuse

Creates or raises a window according to its registered multiplicity policy. Returns the live window record.

Things to Know

key is required for per-key window kinds.

raise_window

raise_window(window_id)
authority: live window focus

Best-effort deiconify, lift, and focus a live window. Calls on_show(record) if present.

close_window

close_window(window_id)
authority: close request

Requests close for a live window. If on_close(record) returns False, the close is canceled. Otherwise it destroys the window.

destroy_window

destroy_window(window_id)
authority: live window destruction

Destroys and unregisters a live window if it exists. Calls on_destroy(record), removes target and instance-key entries, destroys the Toplevel, and removes the window record.

get_window

get_window(window_id)
authority: live window read

Returns a live window record. Raises KeyError if absent.

list_windows

list_windows()
authority: live window read

Returns a list of live window records.

post_event

post_event(target_id, event)
authority: event queue write

Appends an event to a target inbox and logs it. Returns the event object.

post_event_to_window

post_event_to_window(window_id, event)

Alias for post_event(window_id, event).

post_event_to_instance

post_event_to_instance(instance_key, event)
authority: event queue write

Posts an event to the live window associated with an instance key.

post_event_to_kind

post_event_to_kind(window_kind, event)
authority: event queue write

Posts an event to a singleton window kind by computing <window-kind>:singleton.

broadcast_event

broadcast_event(event)
authority: event queue write

Posts a shallow copy of the event to every current target.

register_service_target

register_service_target(target_id, handler, state=None)
authority: target registry

Registers a non-window target with an inbox, handler, state dictionary, and active flag.

Handler Signature

handler(target, event) returns effects or None.

send_message(target_id, message, source_id=None)
authority: message routing

Wraps a payload in a MESSAGE envelope, logs it, and posts it to a target.

send_message_to_window

send_message_to_window(window_id, message, source_id=None)
authority: message routing

Sends a message envelope to a live window target.

send_message_to_instance

send_message_to_instance(instance_key, message, source_id=None)
authority: message routing

Sends a message envelope to a live window found by instance key.

send_message_to_kind

send_message_to_kind(window_kind, message, source_id=None)
authority: message routing

Sends a message envelope to a singleton window kind.

broadcast_message

broadcast_message(message, source_id=None)
authority: message routing

Sends a shallow copy of a message to every current target.

route_effect

route_effect(effect, source_id=None)
authority: effect routing

Routes one reducer or service effect through the runtime. Always appends to rt.effect_log.

Known Effect Types

POST_EVENT, SEND_MESSAGE, SUMMON_WINDOW, CLOSE_WINDOW, DESTROY_WINDOW, REQUEST_PROJECT, LOG, SHOW_MESSAGE_BOX.

route_effects

route_effects(effects, source_id=None)
authority: effect routing

Routes each effect in an iterable. None and empty effect collections are allowed.

start_tick_loop

start_tick_loop()
authority: tick scheduling

Marks runtime running and schedules the next Tk after() tick when root exists.

stop_tick_loop

stop_tick_loop()
authority: tick scheduling

Stops future scheduled tick-loop callbacks and clears rt.g["tick_after_id"].

tick_once

tick_once()
authority: runtime heartbeat

Runs one heartbeat: service queues, window queues, reducers, effects, hooks, projection, and shutdown policy check. Returns the new tick count.

run_ticks

run_ticks(count=1, update_tk=True)
authority: testing helper

Runs tick_once() repeatedly and optionally flushes Tk idletasks and updates after each tick.

toggle_debug

toggle_debug()
authority: debug UI

Flips debug mode. When enabling, summons the runtime dashboard.

get_debug_snapshot

get_debug_snapshot(selected_window_id=None)
authority: debug inspection

Builds a JSON-friendly runtime summary and stores it in rt.debug["last_snapshot"].

copy_debug_snapshot

copy_debug_snapshot(selected_window_id=None)
authority: debug export

Creates an indented JSON debug snapshot and copies it to the Tk clipboard when a root exists.
