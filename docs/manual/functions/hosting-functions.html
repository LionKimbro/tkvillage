Hosting Functions

Project-directory readiness is explicit, small, and friendly to lionscliapp.

  • declare_app
  • ensure_ready
  • run
TkVillage no longer owns application configuration. Use lionscliapp for host app configuration when a TkVillage app runs inside lionscliapp.

lionscliapp Mode

Declare {"lionscliapp": True} and call ensure_ready() at the top of commands that use TkVillage. TkVillage then creates <lionscliapp-project>/tkvillage.

Open the lionscliapp setup walkthrough

Standalone Mode

Declare project-dir-name or project-dir. ensure_ready() creates that directory when the runtime first needs it.
