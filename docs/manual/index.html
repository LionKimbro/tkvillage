TkVillage

TkVillage is a small, inspectable runtime for modular Tkinter apps: a hidden root, summonable Toplevel windows, reducer-owned state, queued semantic events, explicit effects, and a periodic tick loop.

The Runtime Shape

Declare the app with declare_app({...}).
Register window kinds with creation, reducer, and projection functions.
Summon windows. TkVillage creates or raises Toplevels according to the declared policy.
Widget callbacks post semantic events instead of mutating app truth directly.
Each tick drains queues, runs reducers, routes effects, calls hooks, projects dirty windows, and checks shutdown.
Design rule: pass only what varies. TkVillage does not pass the runtime module into callbacks. Import tkvillage.app directly only when a callback genuinely needs runtime inspection.
