Quick Start

One process, one runtime, one hidden Tk root, many summonable Toplevel windows.

1. Declare the Application

Use one dictionary. The shutdown policy is required on purpose; a TkVillage app should say how it exits at the start.

import tkvillage as village

village.declare_app({
    "name": "my-app",
    "project-dir-name": ".my-app",
    "shutdown-policy": "on-last-window-close",
    "shutdown-window-kind": None,
    "on-shutdown": None,
})

2. Register a Window Kind

A window kind is a recipe. It does not mean a window exists yet.

village.register_window_kind(
    "main",
    title="My App",
    multiplicity="singleton",
    create=create,
    make_initial_state=make_initial_state,
    reduce_event=reduce_event,
    project=project,
)

3. Summon and Run

village.summon_window("main")
village.run()
The runtime module is not passed to callbacks. If advanced code needs runtime state, import tkvillage.app directly.
