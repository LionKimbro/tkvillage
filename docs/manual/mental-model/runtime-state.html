Runtime State

TkVillage keeps runtime data in tkvillage.app. Scalars live in g; registries and logs live in explicit module-level dictionaries and lists.

rt.g

g contains scalar-ish runtime values: name, root, host_project_dir, project_dir, project_dir_name, project_root, lionscliapp, is_ready, running flags, tick interval, tick count, shutdown policy, shutdown handler, and log limits.
