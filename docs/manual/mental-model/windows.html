Windows

A window is a live Toplevel plus a window record. The record is the runtime handle that connects Tk widgets, reducer state, event queues, and lifecycle metadata.

Window Record

FieldMeaning
window_idUnique live id, also the target id.
window_kindThe registered kind that produced this window.
instance_keyLookup key for singleton/per-key reuse.
toplevelLive tkinter.Toplevel.
stateReducer-owned semantic dictionary.
event_queueQueued semantic events.
widgetsRuntime-only widget handles.
needs_projectWhether projection should run.

State vs Widgets

State is the debug-friendly truth. Widgets are live toolkit objects. Projection moves state into widgets; reducers should not directly configure widgets.

Use record["widgets"] for Tk objects and record["state"] for semantic data.
